# pt

## 启动
```
npm i
npm run serve
```

### 鼠标
```
点击各个面即可旋转，左右键旋转方向相反
在菜单中选择公式后点击魔方自动执行，右键逆转公式执行
按住魔方后拖动控制魔方视角
悬停于菜单各个按钮提供提示
```
### 快捷键
```
Ctrl + Space 打开/关闭菜单
Ctrl + i 打开/关闭设置面板
Enter 打乱魔方
Shift + Enter 自动复原魔方
Space 魔方视角回正
方向键 精确控制魔方视角
```
